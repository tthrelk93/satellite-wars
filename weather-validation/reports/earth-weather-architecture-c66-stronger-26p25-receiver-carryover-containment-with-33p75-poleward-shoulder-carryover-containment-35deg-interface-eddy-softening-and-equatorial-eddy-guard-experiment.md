## Interpretation

- Architecture C65 showed that C64 is locally useful but globally harmful: the `35°` import branch improves, but the equatorial eddy branch repays that relief too strongly.
- This experiment keeps the active `35°` lane live and changes only the base equatorial softening strength and width, which makes it a clean test of whether the repayment is an equatorial over-softening problem.
- The bounded question is whether a tighter equatorial guard can keep the `35°` import relief while stopping the cross-equatorial sign defect from collapsing further.
